# Product Overview

This workspace is an AWS infrastructure development environment focused on Terraform-based cloud architecture with AI-assisted development through Kiro.

## Purpose
- AWS cloud infrastructure development and management
- Terraform configuration and deployment automation
- AI-assisted AWS architecture design and documentation
- Knowledge retrieval and pricing analysis for AWS services

## Key Features
- Comprehensive AWS service integration via MCP servers
- Terraform infrastructure as code development
- AWS documentation and knowledge base access
- Automated diagram generation for AWS architectures
- Real-time AWS pricing information
- Bedrock knowledge base retrieval capabilities

## Target Environment
- AWS Profile: `default` (o usar variable AWS_PROFILE)
- Primary Region: `us-east-1`
- Focus on infrastructure automation and cloud architecture
